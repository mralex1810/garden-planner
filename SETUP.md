# Инструкция по установке и запуску

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Иконки для PWA

Для полноценной работы PWA необходимо добавить иконки в папку `public/`:
- `icon-192.png` (192x192 пикселей)
- `icon-512.png` (512x512 пикселей)

Можно использовать любые иконки с садовой тематикой или создать простые иконки с эмодзи 🌱.

## Особенности

- Приложение работает полностью офлайн после первой загрузки
- Все данные хранятся локально в IndexedDB
- Можно экспортировать/импортировать данные через файл или QR-код
- Поддерживается установка как PWA на мобильные устройства
