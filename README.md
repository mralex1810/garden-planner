# Garden Planner

Приложение для планирования и управления небольшим семейным огородом (около 20 соток).

## Возможности

- Визуализация участка с размещением объектов (грядки, теплицы, парники, бочки, колодцы, дорожки, кусты, зоны отдыха)
- Управление грядками с указанием типа растения
- История ухода: полив, прополка сорняков
- Планирование будущих посадок
- Советник по посадкам (севооборот, совместимость растений)
- Работа офлайн (PWA)
- Синхронизация данных между устройствами

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Технологии

- React + TypeScript
- Vite
- IndexedDB (Dexie.js)
- Fabric.js (canvas редактор)
- PWA (Service Worker)
